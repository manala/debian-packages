# Debian Packages

## Instructions

Add apt key
```
apt-key adv --recv-keys --keyserver hkp://keyserver.ubuntu.com:80 1394DEA3
```

Add apt source
```
deb [arch=amd64] http://debian.manala.io [stretch|buster|bullseye] main
```
